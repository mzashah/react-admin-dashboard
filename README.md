# React Admin Dashboard

A beautiful, feature-rich admin dashboard built with React, TypeScript, and Tailwind CSS. Includes interactive charts, dark mode, data tables, user management, and more.

![React](https://img.shields.io/badge/React-18-61DAFB?style=for-the-badge&logo=react)
![TypeScript](https://img.shields.io/badge/TypeScript-5-3178C6?style=for-the-badge&logo=typescript)
![Tailwind](https://img.shields.io/badge/Tailwind-3-38B2AC?style=for-the-badge&logo=tailwind-css)

## Features

- 📊 Interactive charts (Recharts)
- 🌙 Dark/Light mode toggle
- 📱 Fully responsive design
- 👥 User management table
- 🔔 Notification system
- 📈 Revenue & analytics widgets
- 🗂️ Sidebar navigation
- 🔐 Auth pages (login/register)
- 📦 Product management
- 🎨 Customizable theme

## Quick Start

```bash
git clone https://github.com/mzashah/react-admin-dashboard
cd react-admin-dashboard
npm install
npm run dev
```

## Tech Stack

- **React 18** with TypeScript
- **Vite** for fast bundling
- **Tailwind CSS** for styling
- **Recharts** for data visualization
- **React Router v6** for routing
- **Zustand** for state management
- **React Table** for data tables
- **Lucide React** for icons

## Pages

- `/` — Overview dashboard with KPIs
- `/analytics` — Detailed analytics
- `/users` — User management table
- `/products` — Product catalogue
- `/orders` — Order management
- `/settings` — Profile & preferences
